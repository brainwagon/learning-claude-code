# learning-claude-code

Concise, opinionated reference notes on Claude Code features — written by someone actively using the tool and documenting what actually matters. Each page trades breadth for density: no padding, no marketing copy, just the syntax, the mental model, and the gotchas. Topics covered include Tasks and Subagents, Git Worktrees, and Billing, with more pages planned on hooks, MCP servers, the memory system, and permissions. Notes are published as both Markdown and styled HTML.
