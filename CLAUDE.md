# CLAUDE.md

This repository contains reference notes on Claude Code features, published as paired Markdown and HTML files.

## Content conventions

- Write new content as a `.md` file first, then generate a matching `.html` file with the same base name.
- HTML files should match the style of existing pages (dark-friendly, monospace font for code, VSCode-style syntax highlighting for code blocks).
- When updating a `.md` file, regenerate its corresponding `.html` file to keep them in sync.
